### The Aim Of The Project
Implementing new technologies and best practices in android development such as:
 - TDD
 - MVVM
 - HILT
 - Navigation Components
 - Coroutines
 - Flows
 - Retrofit
 - LiveData
 - ViewBinding
 - DataBinding
 - Pagination
 - Responsive Design
 - SafeArgs
 - Glide
